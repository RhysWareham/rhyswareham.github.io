description: This is a project that I have been working on recently. When struggling with ADHD, trying to decide the next task to start is a daunting thing, especially when you have an almost endless list of tasks or hobbies that you want to get done; often times I end up getting stuck and doing nothing instead and then feeling disappointed in myself. I decided to create an application where I could put in all of my tasks or things that I want to do, and then allow it to randomly select my next task. I added an urgency level feature too so that tasks can be more or less likely to show up. I then added a timer that can be adjusted by scrolling. Currently, the mechanisms are only laid out for desktop, but I have designed the layout for mobile using Figma. Of course, this entire project may have just been another way of procrastinating from my important tasks whilst still feeling productive XD.
